Function Generator
( kit code-KP 3013)

This function generator uses a wienbridge oscillator for the generic waveform. Other waveforms were achieved using op-amp circuit to perform mathematical functions on the generic waveform.
Supply Voltage +/- 12V
Waveforms: Sine Square, Triangle
Frequency 1KHz (fixed). Please note that the variable frequency version is beyond the scope of projects in this category.
