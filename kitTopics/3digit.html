3Digit Object Counter
(kit code-KM 1065)

This counter uses a breakbeam sensor to count an event, say the number of coca-cola bottles passing through a conveyor belt, or any event suitable for breakbeam counting. The circuit components include comparators, op-amps, timers, counters..etc.
