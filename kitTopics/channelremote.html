4 CHANNEL REMOTE CONTROL
( kit code-KA 1001)

Using this 4-channel infrared remote control, you can control up to 4 different items connected to the mains. The remote serves basically as a power toggle switch which switches ON and OFF the power when triggered.
Range is Approx. 5meters
PIC 16F84 is employed in the cotrol circuitry.
