DIGITAL CODED ACCESS SECURITY SYSTEM
( kit code-KM 5008)

This project (aka, Electronic Code lock) uses an electronic keypad and discrete logic circuits to control access to an electronic device or door.
We have the microcontroller version designed using PIC16F84 (with more sophisticated features employing embedded software technology).
We also have the the computer interface version of this project , where you enter a code via the computer keyboard and the computer controls the opening and closing of a sliding door via a usb port. Software used is VB.NET.
